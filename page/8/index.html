剑指offer之调整数组顺序是奇数位于偶数前面

发表于 2017-02-14 | 分类于 剑指offer | 阅读次数

题目

输入一个整数数组，实现一个函数来调整该数组中数字的顺序，使得所有的奇数位于数组的前半部分，所有的偶数位于位于数组的后半部分，并保证奇数和奇数，偶数和偶数之间的相对位置不变。

思路

看到这个题目，自己一开始也想借鉴之前的排序所用到的方法，但在拿实际例子就行模拟的时候，发现新建一个数组先把奇数放到也给数组，然后再把偶数放到这个数组，最后把这个排序好的数组赋值给原来的数组。这就形成了解法一，该解法时间复杂度为$o(n)$,明显比采用冒泡排序方法的解法二简单一些。剑指offer树上采用从两端进行判断然后交换，这不符合相对位置不变得要求。

机器学习基本概念

发表于 2017-02-06 | 分类于 机器学习 | 阅读次数

机器学习相关分类

机器学习包括：监督学习、非监督学习、半监督学习和强化学习

  • 监督学习：从给定的训练数据集中学习一个模型，是模型能够对任意给定的输入，对其相应的输出做出一个好的预测。监督学习中用到的训练数据集的输出是人用标注的。常见的为分类。
  • 非监督学习：与非监督学习相比，训练数据集没有人为标注的结果，常见的为聚类
  • 半监督学习：介于监督学习和非监督学习之间，训练数据集包含少量的标注数据和非标注数据
  • 强化学习：强调如何基于环境而行动，以取得最大化的预期利益。它并不需要出现正确的输入和输出对，也不需要精确校正次优化的行为，强化学习更专注于在线规划，需要在探索（未知领域）和遵从（现有知识）之间找到平衡。

题目

用两个队列实现一个栈。并且提供pop和push 操作

思路

用两个队列来实现栈的先进后出，队列1用来push数据，队列2用来辅助pop数据。当按照顺序想队列1添加数据，先加进来的在队列的头部，后加进来的在队列的尾部，如果想删除尾部数据，只能把队列1的数据删除并添加到队列2而中，直到队列1中的还剩一个元素，这样就可以删除了。

剑指offer之用两个栈实现队列

发表于 2017-02-04 | 分类于 剑指offer | 阅读次数

问题

用两个栈来实现一个队列，完成队列的Push和Pop操作。 队列中的元素为int类型。

思路

栈是“先进后出”，队列是“先进先出”，用两个栈来实现一个队列，可以想到，两个“先进后出”叠加就变成了“先进先出”。当插入一个元素时，压入stack1。当要删除队列的头的时候，如果stack2为空时，把stack1的元素全部压入stack2，然后删除stack2最上面的一个元素，从而现实了队列的pop操作。
