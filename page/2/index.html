java 中实现多线程有两种方法：继承Thread 类、实现Runnable接口，在程序开发中实现多线程，大多是实现Runnable 接口为主，实现Runnable 接口相比集成Thread类的优势：

  • 可以避免由于java的单继承而带来的局限
  • 增强程序的健壮性，代码能够被多个线程共享，代码与数据是独立的
  • 适合多个相同程序代码的线程区处理同一资源的情况

问题

思路

第一个思路：先对数组排序，然后遍历数组，看到重复的话，就输出，时间复杂度为$O(nlogn)$

第二个思路：构建一个hashMap，存储每一个字符，时间复杂度为$o(n)$,空间复杂度为$o(n)$

第三个思路：根据所有的数字都在0到n-1的范围，遍历的每一个数字的时候，让它和数字对应索引位置进行交换，如果相同则返回
