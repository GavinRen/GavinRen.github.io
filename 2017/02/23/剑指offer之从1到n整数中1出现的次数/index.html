剑指offer之从1到n整数中1出现的次数

发表于 2017-02-23 | 分类于 剑指offer | 阅读次数

问题

求出1~13的整数中1出现的次数,并算出100~1300的整数中1出现的次数？为此他特别数了一下1~13中包含1的数字有1、10、11、12、13因此共出现6次,但是对于后面问题他就没辙了。ACMer希望你们帮帮他,并把问题更加普遍化,可以很快的求出任意非负整数区间中1出现的次数。

思路

最简单最暴力的想法，当时就是从1 到n 循环判断，记下每个数中1的个数。然后计算总和。这个时间复杂度为$O(n*logn)$。第二思路则是去从数字规律招手，这样可以明显提高时间效率。以21345为例。把21345的所有数字分为两段：一段是1到1345，另一段是1346到21345.对这两段分别计算1出现的个数。

1346-21345：

只计算最高位的1时，

  • 当最高位大于1时：当1出现最高位时，10000-19999其他位不论为什么值，至少包含一个1，则该情况下1出现的次数为$10^{n-1}$.
  • 当最高为等于1时：例如12345，最高位为1时，至少包含一个1，则该情况下1出现的次数为12345-10000+1

计算 除了最高位其它位中1的个数

  • 可以把1346-21345 分为：1346-11345和11346-21345。每一段剩下的4位数字中，选择其中一位是1，其余三位可以在0-9这10个数字中任意选择，则总共出现的次数为$2410^3$
